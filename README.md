# AlSawife Factory

A simple GUI application for managing factory operations and invoices.

## Version
1.0.0

## Description
AlSawife Factory is a desktop application built with Python and Flet for managing factory operations, particularly focused on invoice creation and management. The application allows users to create invoices, manage client information, and export data to Excel with proper formatting.

## Features
- Invoice creation and management
- Client ledger management
- Product pricing based on JSON configuration
- Automatic price calculation
- Excel export with professional formatting
- Support for complex pricing structures
- Arabic language support

## Requirements
- Python 3.8 or higher
- See `requirements.txt` for Python package dependencies

## Installation
1. Clone or download this repository
2. Install the required packages: `pip install -r requirements.txt`
3. Run the application: `python main.py`

## Building Executable
To build a standalone executable, run: `python build.py`

## Built with
- [Python](https://www.python.org/) - Programming language
- [Flet](https://flet.dev/) - Framework for building interactive web, desktop and mobile applications
- [XlsxWriter](https://xlsxwriter.readthedocs.io/) - Python module for creating Excel XLSX files
- [OpenPyXL](https://openpyxl.readthedocs.io/) - Python library to read/write Excel xlsx/xlsm files
